A professional color wheel plugin for Adobe Photoshop 2023+ (v24.2+), designed to provide a smooth, SAI/ClipStudioPaint-like color picking experience.

### ğŸš€ Installation Guide

#### Method 1: Automatic (Recommended)
1. Download the latest `.ccx` file from **[Releases](https://github.com/anjaymi/color-wheel-pro/releases)**.
2. Double-click the file. **Adobe Creative Cloud** (or UXP Developer Tools) will open.
3. Follow the prompts to install.
4. Restart Photoshop.
5. Go to menu `Plugins` > `Meow Color Wheel` to open the panel.

#### Method 2: Manual Installation
If the CC installer fails, try this:
1. Rename the installer from `.ccx` to `.zip`.
2. Extract the zip file to get a folder.
3. Copy this folder to the Photoshop Plugins directory:
    *   **Windows:** `C:\Program Files\Common Files\Adobe\Plug-ins\CC\ColorWheelPro3`
    *   **macOS:** `/Library/Application Support/Adobe/Plug-ins/CC/ColorWheelPro3`
4. Restart Photoshop.

### ğŸ® Usage Guide

#### 1. Core Features
*   **ğŸ”’ Lock Brightness**:
    Click the lock icon to maintain the current Value (Brightness) while changing Hue or Saturation. Perfect for shading within the same tone.
*   **ğŸ¨ Color Harmonies**:
    *   **Analogous**: Shows neighboring colors for unified schemes.
    *   **Complementary**: Shows the opposite color for high contrast.
*   **ğŸ‘ï¸ Grayscale Mode**:
    One-click toggle to view the wheel in grayscale. Useful for checking tonal values.
*   **ğŸ“ Shape Toggle**:
    Switch between **Triangle** (better for saturation/value mixing) and **Square** (classic picker) modes.

#### 2. Shortcuts & HUD
*   **HUD Picker**: Hold `Shift + Alt + Right Click` (Windows) / `Ctrl + Option + Click` (Mac) on canvas to bring up Photoshop's native HUD. The plugin syncs bi-directionally.
*   **Numeric Input**: Click on any HSB/RGB value at the bottom to type exact numbers.
