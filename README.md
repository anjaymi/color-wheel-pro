## ğŸ‡¬ğŸ‡§ Meow Color Wheel

A professional color wheel plugin for Adobe Photoshop 2023+ (v24.2+), designed to provide a smooth, SAI/ClipStudioPaint-like color picking experience.

![Banner](public/icon_comp.png)

### âœ¨ Features
*   **Precision Color Wheel**:
    *   **Lock Brightness**: Keep value constant while changing hue/saturation.
    *   **Harmony Rules**: Real-time Analogous and Complementary color guides.
    *   **Customizable Shapes**: Toggle between Square and Triangle wheel types.
*   **Interaction**:
    *   **HUD Picker**: Syncs perfectly with Photoshop's native HUD.
    *   **A/B Palette**: Dual slots for quick color swapping.
    *   **Grayscale Mode**: One-click check for value relationships.
